Requirements
------------
This module requires Drupal 7.X.

Overview:
--------
This module allows you to easily add Google Website Optimizer code in
the appropriate spots on your test and conversion pages.

Installation and configuration:
------------------------------
Simply extract the download package in your modules directory and 
then enable the module in 'admin/build/modules/'. Set up your 
experiments in admin/config/content/google_website_optimizer
